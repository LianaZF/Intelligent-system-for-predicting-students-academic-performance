## Цель проекта

Прогнозирование успеваемости студентов на основе их активности для:
- **Классификации риска** (3 категории)
- **Регрессии точного балла** за экзамен

---

## Используемые признаки

| Признак | Описание | Тип данных |
|---------|----------|------------|
| `hours_studied` | Часы самостоятельной подготовки | Числовой |
| `lms_activity_hours` | Часы активности в системе LMS | Числовой |
| `attendance_percent` | Посещаемость занятий (%) | Процент |
| `previous_scores` | Средний балл за предыдущие сессии | Числовой |

---

## Модели машинного обучения

### Основные модели:
- **Random Forest** (классификация и регрессия)
- **K-Nearest Neighbors** (KNN)
- **Gaussian Naive Bayes**
- **Linear Regression**
- **Voting Ensemble** (ансамбль моделей)
- **CatBoost** (градиентный бустинг)
- **XGBoost** (градиентный бустинг)

### Проверенные гипотезы:
1. **Все 4 признака** → Лучшая точность
2. **LMS + Посещаемость** (2 признака)
3. **LMS + Посещаемость + Прошлые баллы** (3 признака)

## Метрики оценки

### Для классификации:
- **Accuracy** (точность)
- **Recall для класса "риск"** (важная метрика для выявления студентов в зоне риска)

### Для регрессии:
- **R²** (коэффициент детерминации)
- **MAE** (средняя абсолютная ошибка)
- **MSE** (средняя квадратичная ошибка)

## Как использовать

### 1. Обучение моделей:
python src/train.py                    # Все модели
python src/catboost_only_final.py      # Только CatBoost
python src/compare_xgboost_catboost.py # Сравнение моделей
### 2. Прогнозирование:
python src/predict.py                  # Интерактивный режим
### 3. Запуск веб-сервера:
python src/server.py                   # API на порту 8000
### 4. Проверка регрессии:
python src/train_regression.py         # KNN регрессия

## Особенности реализации
Стратифицированное разбиение для сохранения баланса классов
Стандартизация признаков (StandardScaler)
Взвешивание классов для обработки дисбаланса
Ансамблирование моделей (VotingClassifier/Regressor)
Автоматическая генерация отчетов в Excel с форматированием
Визуализация результатов (графики сравнения)

## Интерфейсы
Консольный (predict.py) - для быстрой проверки
Веб-API (server.py) - для интеграции с другими системами
Excel-отчеты - для аналитики и презентаций
